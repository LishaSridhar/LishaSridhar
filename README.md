##  Featured Projects

### [Faithful Explanations for LLM Decisions in Mental Health Conversations](https://github.com/LishaSridhar/xai_project)
Multi-module XAI pipeline showing that the choice of attention aggregation method (raw attention vs. attention rollout) changes the scientific conclusions drawn about LLM explanation faithfulness — validated against human evaluation (κ = 0.80).

### [Suicidal Ideation Detection on Social Media](https://github.com/LishaSridhar/suicide-ideation-detection)
NLP classification system detecting suicidal ideation in Reddit posts — 95.7% recall, 0.948 ROC-AUC — with an interpretable risk-severity triage framework.

### [Spotify Music Popularity Prediction](https://github.com/LishaSridhar/SpotifyMusicPrediction)
ML analysis of Spotify's Global Top 200 charts (2017–2023) showing artist reputation predicts song success better than audio features (XGBoost, ~70% accuracy).
